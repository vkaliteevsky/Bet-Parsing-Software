<date_long_int>|<first_team>|<second_team>|<event_1,...,event_n>|<rate_1,...,rate_n>|<bookmaker_1,...,bookmaker_n>|<yield_vector>|<probability vector>
1. <yield_vector> - вектор доходностей (их может быть произвольное количество, даже 10 штук). По этому вектору нужно будет вычислить агрегированные функции доходностей (напр. среднюю доходность, минимальную доходность и т.п. (см. GUI.txt))
2. <probability_vector> - вектор вероятностей (весов). Количество вероятностей всегда совпадает с количеством доходностей. Сумма вероятностей равна 1.

Методы вычисления агрегированных функций от доходностей. (y1...yn - доходности, p1...pn - вероятности)
1. Средневзевешенная доходность E (это мат. ожидание): E = y1*p1 + y2*p2 + ... + yn*pn
2. Дисперсия средневзвешенной доходности: D_weighted = (1/n)*((y1 - E)^2 + (y2 - E)^2 + ... + (yn - E)^2)
3. Средняя доходность: A = (y1 + y2 + ... + yn) / n
4. Дисперсия средней доходности: D_average = (1/n)*((y1-A)^2 + ... + (yn-A)^2)
5. Минимальная доходность: очевидно
По умолчанию упорядочивание идет по средневзвешенной доходности E
